올인원자동자막 생성기

Whisper AI가 음성을 전사하고, 원문 스크립트와 타이밍을 자동으로 싱크합니다.

⬆ 앱 바로 사용 ⬇ 다운로드
핵심 기능
01

Whisper AI 자동 전사

로컬 Whisper AI로 음성을 전사하고 자연스럽게 분절합니다.

02

스크립트 → 오리지널 자막

스크립트를 자막 형식으로 자동 분절. 글자 수 기준 타임스탬프 자동 부여.

03

원문 스크립트 싱크

Whisper 타이밍 + 오리지널 자막을 자동 매핑. 오타 없는 정확한 자막.

04

무음 구간 제거

호흡, 무음 감지 후 제거. 오디오/영상과 자막이 함께 조정됩니다.

05

다양한 자막 형식

SRT, VTT, ASS, SBV 등 모든 편집 프로그램 호환 형식 지원.

06

완전 무료

구독 없음. 데이터 외부 전송 없음. API 키 불필요.

아래 다운로드 버튼에서 운영체제에 맞는 파일을 받으세요.

→ 다운로드 섹션으로 이동
03

setup 파일 실행

압축 해제 후 실행하면 ffmpeg, Whisper, Flask가 자동 설치됩니다.

Windows: setup.bat 더블클릭
Mac/Linux: bash setup.sh
04

매번 사용할 때마다

컴퓨터를 껐다 켜면 서버가 종료돼요.
다시 사용할 때는 start.bat을 실행하세요.
(setup.bat은 최초 1회만)

Windows: start.bat 더블클릭
Mac/Linux: bash start.sh
→ 실행 후 localhost:5000 접속
05

setup vs start 차이

setup — 최초 1회만. 패키지·ffmpeg 자동 설치.
start — 매번 사용 시. 설치 없이 서버만 즉시 시작.

처음 설치: setup.bat
매일 사용: start.bat

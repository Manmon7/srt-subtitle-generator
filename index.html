핵심 기능
01

Whisper AI
자동 전사

OpenAI Whisper 모델이 음성을 텍스트로 변환합니다. 한국어 최적화, 완전 로컬 실행.

02

원문 스크립트
싱크 병합

Whisper 타이밍과 원문 텍스트를 자동으로 매핑합니다. 오타 없는 정확한 자막 완성.

03

무음 구간
자동 제거

호흡, 무음 구간을 감지하고 제거합니다. 오디오와 자막 싱크가 함께 조정됩니다.

04

한글 10자
자연 분절

한글 글자 수 기준으로 어미, 조사를 고려한 자연스러운 자막 분절.

05

SRT 파일
즉시 다운로드

Premiere Pro, CapCut, DaVinci 등 모든 편집 프로그램에서 바로 사용 가능.

06

완전 무료
오프라인 실행

API 키 없음. 구독 없음. 데이터가 서버로 전송되지 않습니다.

설치 방법
01

Python 설치

python.org에서 Python을 다운로드하세요.
설치 시 반드시 "Add Python to PATH" 체크!

→ python.org/downloads
02

WaveSRT 다운로드

페이지 하단 다운로드 버튼에서 운영체제에 맞는 파일을 받으세요.

→ 하단 다운로드 버튼으로 이동
03

실행

압축 해제 후 setup 파일을 실행하면 자동으로 설치가 완료됩니다.

Windows: setup.bat 더블클릭
Mac/Linux: bash setup.sh
04

브라우저에서 사용

설치 완료 후 브라우저가 자동으로 열립니다.

http://localhost:5000

지금 시작하세요

WaveSRT자동 자막 생성기

Whisper AI가 음성을 전사하고, 원문 스크립트와 타이밍을 자동으로 싱크합니다.

⬆ 앱 바로 사용 ⬇ 다운로드
핵심 기능
01

Whisper AI 자동 전사

로컬에서 완전 실행. API 키, 인터넷 연결 불필요.

02

원문 스크립트 싱크

Whisper 타이밍 + 원문 텍스트 자동 매핑. 오타 없는 자막.

03

무음 구간 제거

호흡, 무음 감지 후 제거. 오디오와 자막이 함께 조정됩니다.

04

텍스트 → SRT

원문 텍스트를 한글 기준으로 자동 분절. 엔터로 직접 편집 가능.

05

SRT 즉시 다운로드

Premiere Pro, CapCut, DaVinci 등 모든 편집 프로그램 호환.

06

완전 무료 · 오프라인

구독 없음. 데이터 외부 전송 없음. 내 PC에서만 실행.

Python 설치

→ python.org/downloads
02

WaveSRT 다운로드

아래 다운로드 버튼에서 운영체제에 맞는 파일을 받으세요.

→ 다운로드 섹션으로 이동
03

setup 파일 실행

압축 해제 후 setup 파일을 실행하면 ffmpeg, Whisper, Flask가 자동 설치됩니다.

Windows: setup.bat 더블클릭
Mac/Linux: bash setup.sh
04

localhost:5000에서 전사 사용

setup이 실행된 상태에서 localhost:5000으로 접속하면 음성 전사 기능을 사용할 수 있어요.
